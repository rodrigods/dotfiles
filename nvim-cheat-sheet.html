📺 Tmux Session Management
CMD
tmux
Start new tmux session
CMD
tmux new -s <name>
Start new named session
CMD
tmux a -t <name>
Attach to named session
CMD
tmux a
Attach to last session
CMD
tmux ls
List all sessions
CMD
tmux kill-session -t <name>
Kill named session
CMD
tmux detach
Detach from current session
